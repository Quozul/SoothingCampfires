# Soothing Campfires Mod for Minecraft

This mod adds a regeneration effect to players standing near a campfire that is cooking food.

## Community & Support

Questions or suggestions? Join the [Discord server](https://discord.gg/M2a9dxJPRy).

## Features
- **Regeneration effect**: Stand near a cooking campfire to recover health over time.
- **Customizable gamerules**:
    - `soothing_campfires:duration`: Duration of the effect in ticks (default: 20)
    - `soothing_campfires:radius`: Effect radius in blocks (default: 5)
    - `soothing_campfires:amplifier`: Effect strength (default: 0)
    - `soothing_campfires:signal_radius`: Effect radius when a hay bale is placed below the campfire (default: 10)

## How to Use
1. Place a campfire
2. Cook food on it
3. Stand within the effect radius to receive regeneration
4. Place a hay bale below the campfire to change the effect radius
